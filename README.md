# Dumas.Tech — Site Institucional

Site institucional da Dumas.Tech. Mais de 10 anos em desenvolvimento de produtos digitais, com especialização em aplicações mobile e soluções para startups, instituições financeiras, grandes organizações e setor público.

## Estrutura do Projeto

```
Company/
├── index.html          # Página principal (HTML semântico)
├── css/
│   └── styles.css      # Todos os estilos (variáveis, componentes, responsivo)
├── js/
│   └── main.js         # Scripts (cursor, partículas, menu, formulário)
└── README.md
```

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript vanilla (Intersection Observer, Canvas)

## Serviços Oferecidos

1. **Aplicações Mobile** — iOS e Android, principal especialização
2. **Soluções Complexas e de Grande Porte** — Enterprise, integrações, segurança
3. **Inteligência Artificial** — Integração em apps e plataformas
4. **Sites e Plataformas Web** — Complemento quando o projeto exige presença multicanal
